# FPC 工坊

FPC（柔性电路板）设计增强工具，专注于金手指区域补强设计、焊盘阻焊处理与电气参数计算。

## 功能

### 压PAD

![压PAD](images/README_press-pad.png)

选中焊盘 → 一键扩展阻焊层压合焊盘，增强 FPC 焊盘结合力。支持 NSMD ↔ SMD 自动转换。

- 焊盘自动识别（RECT / ELLIPSE / OVAL / NGON）
- 阻焊扩展量可调，预处理模式支持先扩焊盘再压阻焊
- 实时预览改动范围

### PI 补强计算器

![PI补强](images/README_pi-stiffener.png)

金手指插拨区 PI 补强厚度精准计算，根据 FPC 总厚、层叠结构、材质类型反向推导标准 PI 厚度。

- **2D 截面可视化**：7 层 FPC 叠构按比例渲染，支持 layer 开关与 hover 交互
- **材质系统**：PI / FR4 / 钢片三种补强材质，各有独立的标准厚度集
- **公差推荐策略**：正公差 +0.03mm 优先（过盈配合）→ 负公差 -0.01mm → 无推荐，自动匹配最优标准厚度
- **双指示器标尺**：🟢 建议厚度（标准值）+ ⬤ 精准厚度（计算值），同一标尺直观对比
- **左侧图例**：颜色标识快速识别标尺各元素含义
- **EDA 联动**：画布选中补强板 → 自动检测铜层 → 一键应用计算值
- **多选批量操作**：表格多选补强板，一键批量写入材质和厚度
- **暗色模式适配**：完整支持 EDA 暗/亮双主题自动切换

### 辅助计算

走线载流 / 线圈电阻 / 过孔电流 快速估算。

## 兼容性

| 平台            | 版本要求   |
| --------------- | ---------- |
| 嘉立创EDA专业版 | ≥ 3.2.152 |
| EasyEDA Pro     | ≥ 3.2.152 |

## 安装

下载 `fpc-workshop_v*.eext`，在嘉立创EDA专业版中通过「扩展 → 本地安装」加载。

## 开发

```bash
npm install
npm run build
```

构建产物位于 `./build/dist/`。
