# 26.6.21

## 新增

1. PI 补强表格支持多选：`selectedStiffener`（单选）→ `selectedStiffeners`（数组），checkbox + 行点击 toggle，批量应用
2. 材质名称中文化 + i18n 基础设施：表格材质列显示（PI / FR4 / 钢片），新增 `SYS_I18n.text()` 调用及硬编码回退
3. 画布选中恢复：应用写回后自动在画布重新选中补强图元（`setDocumentSource` 重渲染导致选中丢失）

## 修复

1. 修复"应用"后表格不刷新：轮询定时器竞态导致旧数据覆盖，改用 `try-finally` 锁守卫全部异步操作
2. 修复深色模式下黑色 / 白色覆盖膜色块不可见：添加白色边框环（`data-theme="dark" .pi-dot { border-color }`）
3. 修复红色"无推荐"气泡尺寸不一致：统一大小与绿色建议气泡相同
4. 修复极限厚度（2.0mm+ PI）时 2D 截面图撑爆整个面板：`html`/`body` 添加 `height:100%; overflow:hidden` 从根约束布局链，`.pi-scene-viewport` 改为 `overflow-y:auto` 承载滚动条

## 变更

1. 引擎最低版本要求：`^2.3.0` → `>=3.2.152`
2. 左侧导航标签优化：PI补强 → 补强厚度，耐电流 → 线路温升，过孔 → 过孔电流
3. 补强表格排序改为层优先（顶层→底层），层内按图元 ID 稳定排序，材质/厚度修改不影响行顺序
